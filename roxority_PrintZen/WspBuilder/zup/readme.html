PrintZen Documentation:

You can access the PrintZen product documentation from the following locations:

  • In your SharePoint after solution installation: via Site Settings / PrintZen Studio / Help Topics
  • Online: at http://roxority.com/PrintZen-Documentation/

PrintZen Installation:

Run either Inst_2013.exe (for SharePoint 2013) or Inst_2010_2007.exe (for SharePoint 2010 and 2007) exactly as described in our help topic and also in our blog.

PrintZen Release Notes

Version 1.3.1 (9th May 2013)

  • New: Configuration setting to embed additional custom CSS files
  • By popular request, the Site Settings / PrintZen Studio section of links is now only visible to administrative users.
  • Updated jQuery version to 1.8.3

Version 1.3 (18th December 2010)

  • Added: integration into SharePoint Site Actions menu
  • Added: support for printing selected Web Part zones
  • Added: support for printing the full page contents
  • Added: support for printing all types of Web Parts (rather than only List Views and PeopleZen)

Version 1.2 (26th September 2010)

  • Added: support for placeholders in custom Web Part page and Print Action title and description
  • Added: support for printing View Item Properties display form (DispForm.aspx) pages

Version 1.1 (25th September 2010)

  • Added: PeopleZen integration for print-friendly SharePoint staff directories, phone listings, departmental or any other User Profile listings
  • Added: Web Part Page for Custom Design of Print View option for Print Actions

Version 1.0 (21st September 2010)

  • Initial release.
