Die folgenden Einstellungen finden administrative Benutzer auf der Seite Konfiguration im PrintZen-Studio, welches Sie über Websiteeinstellungen / PrintZen Studio / Konfigurationseinstellungen erreichen.

Jede Einstellung hat einen globalen, über die gesamte Serverfarm hinweg geltenden Wert, der jeweils optional auf Websitesammlungsebene überschrieben werden kann. Die folgenden Konfigurationseinstellungen stehen zur Verfügung:

Sprache

PrintZen wurde für verschiedene Sprachen lokalisiert. Zusätzlich zu Englisch und Deutsch werden hier alle Sprachen zur Auswahl angeboten, für die Sie eigene Teilübersetzungen (unter Websiteeinstellungen / PrintZen Studio / DIY-Benutzeroberflächen-Lokalisierung) angefertigt haben. Diese Einstellung betrifft alle PrintZen-Benutzeroberflächen außer Beschriftungen und Beschreibungen von Features, Listenvorlagen, Spaltentypen, Inhaltstypen oder Webpartvorlagen (diese werden von SharePoint immer direkt gemäß der aktiven Website-Spracheinstellung geladen).

Wählen Sie aus folgenden Optionen auf der Seite Konfiguration im PrintZen-Studio:

  • (Wie Site, oder Englisch)
  • English
  • Deutsch

Fensteroptionen für Popup-Druckansichten

Sie können für Druck-Ansichten, die in einem neuen Popup-Fenster (statt im aktuellen Browsertab) geöffnet werden, hier die Fensteroptionen für das Popup anpassen. Hinweis: die Syntax hierfür wird durch JavaScript vorgegeben, bei Unklarheiten konsultieren Sie unsere Support-Foren oder Ihren Webentwickler / Skriptspezialisten vor Ort.

Falls Sie diese Einstellung angepaßt haben, können Sie über den Link Auf Standardwert zurücksetzen auf der Seite Konfiguration im PrintZen-Studio folgenden Standardwert wiederherstellen:

width=760,height=560,menubar=1,resizable=1,scrollbars=1,status=0,toolbar=0,location=0,directories=0

URLs zusätzlich einzubindender Cascading Stylesheets (CSS)

Ihre Druckansichten binden zwar bereits automatisch die "Theme"- und "Benutzerdefinierte CSS-URL"-Einstellungen der aktuellen SharePoint-Site ein. Sie können hier jedoch weitere in Ihren Druckansichten zu ladende CSS-Stylesheets angeben (als absolute oder relative URLs, eine pro Zeile).

Pfad zum Merge-Werkzeug

Geben Sie die vollen Dateipfad (inklusive Dateinamen) zu Ihrer doc2pdf.exe-Programmdatei oder äquivalenten Programm an. HINWEIS: in Multi-Server-Farmumgebungen sollte das Programm auf allen Farmservern an exakt demselben Pfad vorliegen.

PrintZenServer.exe kann WEDER serverseitiges Dokumentmerging NOCH serverseitiges Drucken unterstützen, solange hier keine gültige und voll funktionsfähige doc2pdf.exe oder äquivalente Programmdatei angegeben wird.

Heften (Stapling)

Gibt an, ob und an welcher Position ein Drucker ein mehrseitiges Dokument heftet.

Wählen Sie aus folgenden Optionen auf der Seite Konfiguration im PrintZen-Studio:

  • Keine Heftung
  • Oben links
  • Oben rechts
  • Unten links
  • Unten rechts
  • Rückendrahtheftung
  • Zweifach links
  • Zweifach rechts
  • Zweifach oben
  • Zweifach unten
  • Nicht angegeben bzw. Drucker-Voreinstellung oder -Standardeinstellung

Beidseitigkeit (Duplex)

Gibt an, ob ein Drucker einseitigen Druck oder einen zweiseitigen Drucktyp (Duplexdruck) verwendet.

Wählen Sie aus folgenden Optionen auf der Seite Konfiguration im PrintZen-Studio:

  • Einseitig
  • Beidseitig (langer Rand)
  • Beidseitig (kurzer Rand)
  • Nicht angegeben bzw. Drucker-Voreinstellung oder -Standardeinstellung

Keine Standardinstanzen für PrintZen-Konfiguratoren auf Websitesammlungsebene (NICHT empfohlen!)

Konfiguratoren sind Listen von Elementen, die das Verhalten dieser Software definieren bzw. anpassen (wie Export-Aktionen in ExportZen, Datenquellen und Datenfeldvorlagen in PeopleZen, Druckansichten und Druck-Launcher in PrintZen, Upload-Aktionen in UploadZen). Standardmäßig besitzt jede Websitesammlung für jeden Konfigurator ihre eigene, nicht entfernbare Standardinstanz , die angepaßt werden kann oder zu der weitere hinzugefügt werden können. In sehr großen Serverfarmen mit 1000en von Websitesammlungen können diese Standardinstanzen eher hinderlich sein und der ausschließliche Einsatz von Instanzen auf Farmebene präferiert werden. Kreuzen Sie diese Option an, um die Standardinstanz-pro-Websitesammlung zu deaktivieren.

Standardmäßig keine Einbindung der jQuery-Bibliothek v1.8.3 (NICHT empfohlen)

Einige PrintZen-Benutzeroberflächen erfordern die jQuery-Bibliothek und binden diese in die entsprechenden Seiten ein. Mehrfacheinbindung von jQuery kann manchmal zu Problemen führen. Kreuzen Sie auf der Seite Konfiguration im PrintZen-Studio diese Option an nur, wenn Ihre Masterseiten, Webpartseiten oder Drittanbietererweiterungen eigenständig, global (nicht nur in Einzelfällen) und unabhängig von PrintZen ebenfalls jQuery einbinden, um PrintZen davon abzuhalten, jQuery einzubinden. Ist diese Option aktiviert, müssen Sie selbständig sicherstellen, dass jQuery (in einer Version neuer als 1.4 ABER älter als 1.9) in Seiten geladen wird, die PrintZen-Benutzeroberflächen enthalten, andernfalls wird PrintZen unter Umständen nicht mehr funktionieren. Dies ist eine globale Voreinstellung, die lokal ggf. noch einmal individuell überschrieben werden kann.
