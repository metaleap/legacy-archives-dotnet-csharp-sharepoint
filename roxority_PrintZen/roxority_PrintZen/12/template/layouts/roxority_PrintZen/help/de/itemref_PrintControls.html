Die folgenden Einstellungen können für jede Druck-Launchersammlung durch administrative Benutzer auf der Seite Druck-Launcher in Ihrem PrintZen Studio vorgenommen werden; via Websiteeinstellungen / PrintZen Studio / Druck-Launcher:

Register "Allgemein":

Name dieser Druck-Launchersammlung

(Texteingabefeld)

Dieser Name wird üblicherweise Endanwendern nirgends angezeigt, hilft Ihnen aber gegebenenfalls, die Übersicht zu behalten.

Diese Druck-Launchersammlung ist...

(Definierung einer Untermenge an SharePoint-Websites)

Optional: wählen Sie aus, in welchen Websites diese Druck-Launchersammlung verfügbar sein wird. Standardmäßig steht Sie allen Websites zur Verfügung und dies muß normalerweise bei der Ersteinrichtung dieser Software auch vorerst nicht geändert werden. Nehmen Sie nur Änderungen vor, wenn Sie mit der in obigen Steuerelementen verwendeten Terminologie vertraut sind. Sollten Ihre Änderungen nicht die erwarteten Wirkungen erzielen, können Sie diese jederzeit auf "in allen Websites ohne Ausnahmen verfügbar" zurücksetzen und erneut inkrementell mit Ihren Ausnahmen experimentieren.

Um eine Menge an SharePoint-Websites zu beschreiben, wählen Sie zunächst entweder verfügbar in allen Websites oder verfügbar in keinen Websites, und klicken Sie dann auf den Link außer..., um nun zu definieren, welche Websites aus dieser Vorauswahl ausgeschlossen werden sollen. Ausgeschlossen werden können:

  • die Stammwebsite (der jeweils aktuellen Websitesammlung)
  • Websites auf Basis der ausgewählten Sitedefinitionen
  • oder Websites mit den angegebenen Titeln, URL-Pfadfragmenten oder IDs.

Beschriftung für Druck-Launcher

(Texteingabefeld)

Geben Sie die Beschriftung dieser Druck-Launcher an, die in Websiteaktionen-Menüs, die diese Druck-Launcher verwenden, angezeigt werden soll. (Geben Sie keine Beschriftung an, so wird der oben angegebene Name für diese Druck-Launcher verwendet, oder falls ebenfalls leer, ihre interne ID.)

Standardwert für neue Druck-Launcher:

$Resources:Tool_PrintControls_DefTitle

Beschreibung für Druck-Launcher

(Texteingabefeld)

Geben Sie die Beschreibung dieser Druck-Launcher an, die in Websiteaktionen-Menüs, die diese Druck-Launcher verwenden, angezeigt werden soll.

Register "Verfügbarkeit und Druckbefehle":

In "Websiteaktionen" anbieten (erfordert aktives Websitesammlungsfeature)

(Ankreuzbares Kontrollfeld)

Falls angekreuzt (und bei aktiviertem Websitesammlungsfeature PrintZen Menüintegration) werden alle von dieser Druck-Aufrufsammlung angebotenen Druckbefehle in ihrem eigenen Untermenü (welches die oben angegebene Beschriftung und Beschreibung anzeigt) innerhalb des SharePoint-eigenen Websiteaktionen-Menüs angeboten.

Drucken des Seiteninhaltsbereichs anbieten, wenn möglich

(Ankreuzbares Kontrollfeld)

Falls angekreuzt, bietet dieser Launcher eine Druck-Ansicht für den gesamten Seiteninhalt (ohne Seitenchrom- und Navigationsartefakte wie Ribbon, Top-Navigation, Schnellstartleiste etc.) an.

Drucken einzelner Webparts anbieten, wenn möglich

(Ankreuzbares Kontrollfeld)

Falls angekreuzt, bietet dieser Launcher individuelle Druck-Ansichten für jedes einzelne Webpart auf der aktuellen Seite an.

Drucken einzelner Webpartzonen anbieten, wenn möglich

(Ankreuzbares Kontrollfeld)

Falls angekreuzt, bietet dieser Launcher individuelle Druck-Ansichten für jede einzelne Webpartzone auf der aktuellen Seite an.
