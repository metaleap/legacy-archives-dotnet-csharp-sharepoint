Settings for Print Launchers

The following settings can be customized for each Print Launcher by administrative users on the Print Launchers page in your PrintZen Studio, accessible via Site Settings / PrintZen Studio / Print Launchers:

"General" tab:

Name for this Print Launcher

(Text box)

This name will typically not be visible on content pages to end users, but helps you stay organized.

This Print Launcher is...

(Descriptor for a set of SharePoint Web Sites)

Optionally select in which web sites this Print Launcher will be available. By default, it will be available in all web sites and this usually does not need to be changed during your first explorations within this software. Only perform changes if you are familiar with the terminology presented in the above property editor. Should your changes not result in the expected outcome, you can always easily set this back to "all web sites, no exceptions" and then resume experimenting incrementally with your exceptions.

To define the set of Web Sites, first select either Available in all Web Sites or Available in no Web Sites, then click the except... link to describe which web sites to exclude from that selection. You can exclude:

  • the root Web Site (of the respectively-current Site Collection)
  • Web Sites based on the selected site definitions
  • or Web Sites with the specified titles, URL path fragments or IDs.

Print Launcher caption

(Text box)

Specify the caption for this Print Launcher to be shown in the Site Actions menus that use this Print Launcher. (If you do not specify a caption, the above name for this Print Launcher will be used, or if blank, its internal ID.)

Default value for new Print Launchers:

$Resources:Tool_PrintControls_DefTitle

Print Launcher description

(Text box)

Specify the description for this Print Launcher to be shown in the Site Actions menus that use this Print Launcher.

"Availability and Print Commands" tab:

Show in "Site Actions" menu (requires activated Site Collection feature)

(Check box option)

If ticked (and if the PrintZen Menu Integration Site Collection feature is activated), all available print commands provided by this Print Launcher appear in its own sub-menu (showing the caption and description specified above) inside the SharePoint Site Actions menu.

Offer printing of the page content area, if possible

(Check box option)

If ticked, this Print Launcher offer a command to print the content area of the current page, excluding out-of-box page chrome and navigational artifacts such as ribbon, top navigation, quick launch etc.

Offer printing of individual Web Parts, if possible

(Check box option)

If ticked, this Print Launcher offers Print commands for each Web Part on the page.

Offer printing of individual Web Part zones, if possible

(Check box option)

If ticked, this Print Launcher offers a command to print all Web Parts in a selected zone only.
