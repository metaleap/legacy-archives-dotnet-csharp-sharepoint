Version 1.3.1 (8th May 2013)

  • By popular request, the Site Settings / PrintZen Studio section of links is now only visible to administrative users.
  • Updated jQuery version to 1.8.3

  • Added: integration into SharePoint Site Actions menu
  • Added: support for printing selected Web Part zones
  • Added: support for printing the full page contents
  • Added: support for printing all types of Web Parts (rather than only List Views and PeopleZen)

  • Added: support for placeholders in custom Web Part page and Print View title and description
  • Added: support for printing View Item Properties display form (DispForm.aspx) pages

Version 1.1 (25th September 2010)

  • Added: PeopleZen integration for print-friendly SharePoint staff directories, phone listings, departmental or any other User Profile listings
  • Added: Web Part Page for Custom Design of Print View option for Print Views
