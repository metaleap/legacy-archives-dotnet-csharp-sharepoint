LookupZen Documentation:

You can access the LookupZen product documentation from the following locations:

  • Offline: In the docs directory, open the LookupZen-Help.html file in a Web Browser of your choice.
  • In your SharePoint after solution installation: at http://localhost/_layouts/roxority_LookupZen/default.aspx?doc=intro
    • If the above link does not work after successfully completed solution installation, try replacing localhost in the URL with your SharePoint server name / domain name / machine name / virtual host name.
  • Online: at http://roxority.com/lookupzen-documentation
  • Offline: In the docs directory, open the LookupZen-Help.chm Windows HTML Help file.
    • If you cannot open the LookupZen-Help.chm Windows HTML Help file or if it does not display any content, then your Windows security settings prevent the file from being displayed properly because it was downloaded from the Web. You can fix this by right-clicking the file in Windows Explorer, selecting Properties and then clicking the Unblock button on the General tab in the Properties dialog box.
