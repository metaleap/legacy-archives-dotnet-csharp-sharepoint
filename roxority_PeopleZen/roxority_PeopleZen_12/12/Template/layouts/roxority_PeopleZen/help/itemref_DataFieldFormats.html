The following settings can be customized for each Data Field Template by administrative users on the Data Field Templates page in your PeopleZen Studio, accessible via Site Settings / PeopleZen Studio / Data Field Templates:

Name for this Data Field Template

(Text box)

This name will typically not be visible on content pages to end users, but helps you stay organized.

Content mode

(Drop-down menu to pick from a set of choices)

Keep this set to Plain if you are only providing textual content with or without placeholders. Set this to HTML if you need to provide additional imagery, custom formattings, hyperlinks etc. Both modes support [placeholders in square brackets].

Note: in caching-enabled Data Sources using this Data Field Template, changes made to this setting will not be reflected immediately but delayed as configured on the Data Source's Caching tab.

  • Plain text
    — content will be HTML-encoded
  • HTML source code
    — will be rendered directly by the browser

Template content

(Text box)

Specify the content for this template. You can use placeholders in [square brackets] for referencing other data field values. Use [1], [2], [3] .. [n] to reference data values passed to this template.

Note: in caching-enabled Data Sources using this Data Field Template, changes made to this setting will not be reflected immediately but delayed as configured on the Data Source's Caching tab.
