Die folgenden Einstellungen können für jede Datenfeldvorlage durch administrative Benutzer auf der Seite Datenfeldvorlagen in Ihrem PeopleZen Studio vorgenommen werden; via Websiteeinstellungen / PeopleZen Studio / Datenfeldvorlagen:

Name dieser Datenfeldvorlage

(Texteingabefeld)

Dieser Name wird üblicherweise Endanwendern nirgends angezeigt, hilft Ihnen aber gegebenenfalls, die Übersicht zu behalten.

Inhaltsmodus

(Dropdown-Menü zur Auswahl einer Option)

Auf Klartext belassen, wenn nur Textinhalte (mit oder ohne Platzhaltern) bereitgestellt werden. Setzen Sie dies auf HTML, wenn zusätzlich Bilder, Links oder benutzerdefinierte Formatierungen dargestellt werden sollen. Beide Modi unterstützen [Platzhalter in eckigen Klammern].

Hinweis: in Datenquellen, die diese Datenfeldvorlage verwenden und Caching aktiviert haben, werden Änderungen an dieser Einstellung nicht sofort, sondern verzögert (gemäß der Caching-Konfiguration der Datenquelle) erscheinen.

  • Klartext
    — Inhalt wird in korrektes HTML konvertiert
  • HTML-Quellcode
    — wird unverarbeitet an den Browser weitergegeben

Vorlageninhalt

(Texteingabefeld)

Geben Sie den Inhalt dieser Vorlage an. Sie können auf andere Datenfeldwerte mittels Platzhaltern in [eckigen Klammern] verweisen. Verwenden Sie [1], [2], [3] .. [n] um auf übergebene Datenwerte zu verweisen.

Hinweis: in Datenquellen, die diese Datenfeldvorlage verwenden und Caching aktiviert haben, werden Änderungen an dieser Einstellung nicht sofort, sondern verzögert (gemäß der Caching-Konfiguration der Datenquelle) erscheinen.
