How to upload multiple files:

  • Navigate to a directory containing files for upload by clicking the Change Folder... button.
  • "Mark for upload" each file to be uploaded by:
    • either double-clicking the file, or
    • selecting multiple files and clicking the Mark selected file(s) for upload button.
  • Repeat the above steps as needed, then once you're ready to upload, click the Upload Files button.

Can't see a file you know is there?

  • Open the File Filters drop-down menu and if a file filter is active, select (No File Filters) to show all files.

How to bulk-upload entire folders:

  • Navigate to the directory to be uploaded by clicking the Change Folder... button.
  • Now open the Change Folder... drop-down menu and drop-down menu and click the Mark entire folder contents for upload menu item.
  • Repeat the above steps as needed, then once you're ready to upload, click the Upload Files button.
