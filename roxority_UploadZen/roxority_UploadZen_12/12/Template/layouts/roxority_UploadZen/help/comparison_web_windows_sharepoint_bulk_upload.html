All UploadZen Upload Actions technically support two uploader modes:

  • Web Uploader — the default, recommended, "new" uploader mode supporting the widest range of client browsers and operating systems.
  • Windows Uploader — the "old", deprecated mode included and supported only for legacy UploadZen installations. For new installations, Web Uploader is the recommended mode.
