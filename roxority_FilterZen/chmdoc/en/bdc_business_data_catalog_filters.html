Business Data Catalog (BDC) Filters

Note: this FilterZen filter type requires Microsoft Office SharePoint Server (MOSS) 2007 Enterprise Edition with the Business Data Catalog (BDC) features enabled, and the FilterZen BDC Extensions installed.

Create a Business Data Catalog (BDC) filter to send filters based on any data stored in any data sources integrated into your MOSS 2007 Business Data Catalog. Whether you let users pick an entity instance to use in interactive mode, or specify the record to use directly, the filter value being sent is always the current, live value from the specified data source.

To connect to your data source, you select the BDC application and data type to use. Now you simply select which column values are displayed to users in interactive mode, and which column values are sent as filter values to connected Web Parts. You may also want to set any of the general filter settings that are naturally also available to your BDC filters.
