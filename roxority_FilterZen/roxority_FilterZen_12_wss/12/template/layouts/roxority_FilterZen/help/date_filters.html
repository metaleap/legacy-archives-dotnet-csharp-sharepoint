Create a date filter to let users enter or pick a date filter value, or to send a filter value that is always based on the current date. You may also optionally specify a number of days to always add to or subtract from the date filter value just before sending it, so as to easily accomplish relative date-filtering concepts such as yesterday / tomorrow, one week ago / next week etc.

Special Date Values

  • today — this filter value always returns the current date
  • lastmonth_firstday — this filter value always returns the first day of the last month
  • lastmonth_lastday — this filter value always returns the last day of the last month
  • thismonth_firstday — this filter value always returns the first day of the current month
  • thismonth_lastday — this filter value always returns the last day of the current month
  • nextmonth_firstday — this filter value always returns the first day of the next month
  • nextmonth_lastday — this filter value always returns the last day of the next month
