RollupZen Documentation:

You can access the RollupZen product documentation from the following locations:

  • Offline: In the docs directory, open the RollupZen-Help.html file in a Web Browser of your choice.
  • In your SharePoint after solution installation: via Site Settings / RollupZen Studio / Help Topics
  • Online: at http://roxority.com/RollupZen-Documentation/
  • Offline: In the docs directory, open the RollupZen-Help.chm Windows HTML Help file.
    • If you cannot open the RollupZen-Help.chm Windows HTML Help file or if it does not display any content, then your Windows security settings prevent the file from being displayed properly because it was downloaded from the Web. You can fix this by right-clicking the file in Windows Explorer, selecting Properties and then clicking the Unblock button on the General tab in the Properties dialog box.
